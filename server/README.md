

# bigtodo-server
This is a pomotodo app using these technologies:
- TypeScript
- Next.js
- GraphQL
- SSR
- React Hooks

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
